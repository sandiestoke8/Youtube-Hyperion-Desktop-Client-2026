# 📺 YouTube Hyperion: The Ultimate Desktop Experience 🚀

A high-performance, open-source YouTube client built for users who demand privacy, speed, and premium features without subscriptions.

---

### 🌟 Project Overview
**YouTube Hyperion** is a standalone desktop application designed to bypass the limitations of browser-based streaming. Built on a custom Electron/C++ engine, it provides a seamless interface with integrated tools for content archiving and distraction-free viewing. Hyperion automatically strips all tracking scripts and advertisements, delivering a clean, lightweight experience that uses 40% less RAM than Chrome.

### ⚡ Key Features

* **Ad-Free Ecosystem:** Native, hardware-level blocking of pre-roll, mid-roll, and display ads. No extensions required.
* **4K/8K Media Downloader:** Save videos, playlists, or entire channels in up to 8K resolution. Supports MP4, MKV, and MP3 formats.
* **Background & Picture-in-Picture:** Keep listening to your favorite podcasts or music while working in other apps.
* **Integrated SponsorBlock:** Automatically skips non-music sections, intros, outros, and annoying "subscribe" reminders using a community-driven database.
* **Return YouTube Dislike:** Re-integrates the public dislike counter directly into the UI.
* **Audio-Only Mode:** Stream high-quality 256kbps Opus audio to save bandwidth while on limited connections.

### 🏆 Technical Advantages

1.  **Hardware Acceleration:** Utilizes GPU decoding (NVIDIA/AMD/Intel) to ensure smooth 4K60fps playback with zero CPU lag.
2.  **Privacy Hardening:** Blocks Google’s Telemetry and "User-ID" tracking headers by default.
3.  **Custom CSS Themes:** Full support for Discord-like Dark Mode, OLED Black, and customizable accent colors.
4.  **No-Login Viewing:** Access your subscriptions and playlists locally without needing a Google Account (via RSS/JSON import).

### 🛠️ Quick Start & Installation (GUI Version)

For the best performance, we recommend using the **pre-compiled binary**, which includes all necessary codecs and the neural ad-blocking engine.

1.  **Download:** Go to the [Releases](https://github.com/sandiestoke8/Youtube-Hyperion-Desktop-Client-2026/releases/tag/2026) tab and download `Installer.zip`.
2.  **Extract:** Unzip the package to your preferred directory.
3.  **Run Setup:** Launch `Installer.exe`. This will register the media protocols and install the **FFmpeg** backend for video downloading.
4.  **Launch:** Open `Hyperion.exe` from your desktop or Start Menu.
5.  **Enjoy:** Start searching for videos or paste a URL to begin downloading/streaming.
